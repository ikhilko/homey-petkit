# Petkit Homey App

Control and monitor your Petkit devices from Homey.

## Features

- Connect to Petkit devices
- Monitor device status
- Control device operations
- Flow card support

## Installation

1. Install the app from the Homey App Store
2. Configure your Petkit account credentials in the app settings
3. Add your Petkit devices

## Supported Devices

This app supports various Petkit devices including:
- Smart feeders
- Water fountains
- Air purifiers
- Pet carriers

## Development

To develop this app locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Run with Homey CLI: `homey app run`
